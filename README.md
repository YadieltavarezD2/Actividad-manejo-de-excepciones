# Aplicación de registro de estudiantes con manejo de excepciones


# Descripcion:
aplicación de escritorio utilizando en C#
que permita registrar información básica de un usuario y validar los datos
ingresados mediante el uso de manejo de excepciones.

# Herramientas utilizadas:

C#

.NET Framework

Visual Studio 2022

# Uso o Ejecución:


1. Clonar el repositorio desde GitHub

```  bash
   
   git clone https://github.com/YadieltavarezD2/Actividad-manejo-de-excepciones
```
 
3. Abrir el archivo `.sln` en Visual Studio

4. Compilar el proyecto

5. Ejecutar la aplicación presionando **F5** o el botón **Iniciar**

6. La aplicación se abrirá mostrando el formulario principal
